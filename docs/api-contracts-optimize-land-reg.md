# API Contracts: Optimize Land Reg

## Overview
This document contains the discovered API endpoints for the main `optimize-land-reg` module.

## Endpoints
- **ActorController**: Manages actors (persons, public/private entities).
- **FindingController**: Manages findings/observations.
- **FingerprintMatchingHistoryController**: History of fingerprint matchings.
- **SynchroHistoryController**: Tracks synchronizations from mobile devices.
