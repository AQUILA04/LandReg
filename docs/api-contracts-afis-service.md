# API Contracts: AFIS Service

## Overview
This document contains the discovered API endpoints for the `afis-service` microservice.

## Endpoints
- **FingerprintStoreResource**: Handles fingerprint storage APIs.
- **MatcherJobHistoryResource**: Tracks background matching jobs.
- **PublicUserResource**: Manages user endpoints.
- **AfisServiceKafkaResource**: Kafka endpoints for asynchronous processing.
