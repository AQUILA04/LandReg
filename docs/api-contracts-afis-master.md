# API Contracts: AFIS Master

## Overview
This document contains the discovered API endpoints for the `afis-master` module.

## Endpoints
- **PublicUserResource**: Manages public user interactions.
- **ProcessingFingerprintResource**: Handles processing of fingerprints.
- **MatcherJobHistoryResource**: Manages matcher job history records.
- **FingerprintStoreResource**: Handles fingerprint storage interactions.
- **AfisMasterKafkaResource**: Kafka messaging endpoints.
